### Overview
A high-performance community hub and game library optimized for SFUSD Chromebook performance. This project serves as a central dashboard for educational tools, custom systems, and simulation gaming.

Key Features
Game Library: Access to various simulation and sandbox titles like Eaglercraft, Papa's Freezeria, and Bee Swarm.

Stealth Mode: A built-in research overlay with custom presets for Math, Science, and History to maintain a professional appearance in educational environments.

Theming Engine: Support for custom Minecraft and K-pop aesthetic themes.

Diagnostics: Real-time system status dashboard to track network performance and asset availability.
